## 🚀 Quick Start

Getting started is as easy as 1-2-3! No complex setup required.

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for game loading

### Installation & Usage
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Dhanam-patel/Arcadia-X.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd Arcadia-X
   ```

3. **Open the website**:
   - Simply open `index.html` in your preferred web browser
   - Or use a local server for the best experience:
     ```bash
     # Using Python (if available)
     python -m http.server 8000

     # Or using Node.js (if available)
     npx serve .
     ```

4. **Explore!** 🕹️
   - Navigate through the pages using the header menu
   - Click on game buttons to play embedded games
   - Check out the About section for some laughs
   - Connect via the Contact page

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)

- **Fonts**: Custom fonts from CDN Fonts (Federico, Play Time, Awakenning, Frozito)
